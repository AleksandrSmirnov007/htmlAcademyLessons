Верстка CSS
Вы можете расширить нашу стандартную конфигурацию до lint CSS.

1. Используйте npm и наш init инструмент для установки Stylelint и конфигурации:

npm init stylelint

2. Запустите Stylelint для всех CSS-файлов в вашем проекте:

npx stylelint "**/*.css"

Для того чтобы удалить NPM-пакет из проекта, нужно прописать следующую команду:
npm uninstall myPackage.
Данная команда удалит как сам пакет, так и все зависимости необходимые для его работы. Пакет будет удален как из файла package.json (из dependencies и devDependencies соответственно), так и из директории node_modules будет удалено все связанное с этим пакетом.

# Как сгенерировать файл package -lock.json без запуска npm install
Используйте npm install --package-lock-only команду для создания package-lock.json файла без запуска npm install.

--package-lock-only Опция только обновляет package-lock.json файл вместо проверки вашей node_modules папки и загрузки зависимостей.

Откройте свой терминал в корневом каталоге вашего проекта (где находится ваш package.json файл) и выполните следующую команду:  npm install --package-lock-only

Когда установлен аргумент --package-lock-only , создается package-lock.json файл, если он еще не существует.

Если файл существует, он обновляется.

Обратите внимание, что команда не проверяет вашу node_modules папку и не загружает зависимости.
