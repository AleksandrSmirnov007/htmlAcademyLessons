5.6. Чек-лист самопроверки

Использованы собственные классы WordPress.

Для меню создана разметка меню WordPress:

у основного меню создан  < nav >  и дочерний div с классом menu-primary-container,
у < ul > добавлен класс menu,
у пунктов списка класс menu-item,
у ссылок нет классов,
у пункта списка текущей страницы класс current-menu-item,
у подменю прописан класс sub-menu,
у пунктов списка с подменю добавлен класс menu-item-has-children,
у других меню создан < nav > и дочерний div с классом вида menu-названиеменю-container, например, menu-footer-container.
Для Bitrix использованы любые классы для создания меню.
Пример разметки:
  • Каталог
    • Новинки
    • Сливочное
    • Щербеты
    • Фруктовый лед
    • Мелорин
  • Доставка и оплата
  • О Компании
